description: Quick capture an idea to ideas.md
argument-hint: [idea description]
---

Add a new idea to ideas.md with:

**Title/Description:** $ARGUMENTS

**Instructions:**
1. Read ideas.md to find the highest IDEA-XXX number
2. Create the next sequential ID (e.g., if IDEA-010 exists, use IDEA-011)
3. Insert the new idea BEFORE the template (IDEA-001) at the bottom
4. Use this format:

```
## IDEA-XXX: [Title from user's description]
**Status:** new
**Added:** [today's date YYYY-MM-DD]

[User's description as one-liner]

### Details
[Leave empty or ask user if they want to add more]

### Notes
- [Any initial thoughts]

---
```

5. Confirm what was added and the ID assigned
6. Ask if user wants to add details or notes
